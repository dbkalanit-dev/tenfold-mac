# Product Roadmap

## Phase 1: Foundation (Completed)
[x] Clipboard History MVP: Persistent capture and retrieval system.

[x] Background Monitoring: MenuBarExtra integration with native macOS accessibility APIs.

[x] Conflict Resolution: Engineered "ignore-self" filter to resolve event-loop conflicts.

## Phase 2: Security & Privacy (Immediate)
[ ] Privacy Filter: Implement detection logic to exclude passwords and sensitive data from history.

[ ] Sensitive Data Scrubbing: Regex-based sanitization for PII (Personally Identifiable Information).

[ ] Data Encryption: Implement Keychain storage for sensitive clipboard snippets.

[ ] Unit Testing: Implement testing for clipboard parsing and filter logic.

## Phase 3: Advanced UX & Distribution (Vision)
[ ] Configurable Hotkeys: Use UserDefaults to allow user-defined key bindings.

[ ] Rich Text/Image Support: Extend NSPasteboard protocols to handle RTF and images.

[ ] Analytics Dashboard: Visualization of clipboard usage patterns.

[ ] App Store Distribution: Sandbox integration, notarization, and final public release.
